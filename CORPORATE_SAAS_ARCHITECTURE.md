# Corporate SaaS Architecture for Bestemail

## Project Overview
Building a enterprise-grade email marketing platform that competes with Mailchimp, Brevo, and HubSpot.

## Technology Stack
- **Frontend**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS + Custom Design System
- **Database**: Supabase (PostgreSQL)
- **Email Service**: AWS SES + Sendy
- **Analytics**: Plausible Analytics
- **CDN**: Cloudflare
- **Security**: NextAuth.js, CSP Headers, Rate Limiting
