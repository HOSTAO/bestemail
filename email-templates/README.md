# Email Templates for Bestemail Platform

Ready-to-use email templates designed for Indian businesses. Each template includes copy-paste HTML and tips for customization.

## Templates Included

1. **welcome-email.html** ✅ - Engaging welcome email for new subscribers
2. **diwali-offer.html** ✅ - Festival promotion template for Diwali sales
3. **order-confirmation.html** ✅ - Clean order confirmation with order summary + tracking CTA
4. **abandoned-cart.html** ✅ - Recovery email with urgency bar + optional discount code
5. **newsletter-simple.html** ✅ - Two-section newsletter with highlight box + CTA
6. **re-engagement.html** ✅ - Win-back email with what's new grid + soft unsubscribe exit

## Usage

1. Copy the HTML from any template
2. Replace placeholder text with your content
3. Update colors to match your brand
4. Test on both mobile and desktop

## Template Variables

All templates support these dynamic variables:
- `{{first_name}}` - Subscriber's first name
- `{{company_name}}` - Your company name
- `{{unsubscribe_link}}` - Unsubscribe URL
- `{{product_name}}` - Product name (for e-commerce)
- `{{product_price}}` - Product price
- `{{discount_code}}` - Promotional code
- `{{discount_percent}}` - Discount percentage
- `{{order_id}}` - Order number
- `{{order_total}}` - Total order amount
- `{{order_tracking_link}}` - Order tracking URL
- `{{shipping_cost}}` - Shipping charge (use "FREE" if applicable)
- `{{delivery_address}}` - Customer delivery address
- `{{estimated_delivery}}` - Estimated delivery date
- `{{support_whatsapp}}` - Your WhatsApp support number
- `{{cart_recovery_link}}` - Abandoned cart recovery URL
- `{{newsletter_subject}}` - Newsletter headline
- `{{newsletter_date}}` - Date of issue
- `{{reengagement_link}}` - Link to bring inactive users back
- `{{offer_expiry}}` - Offer expiry text (e.g. "7 days")

## Best Practices

- Keep subject lines under 50 characters
- Place main CTA above the fold
- Use alt text for all images
- Test links before sending
- Include unsubscribe link in footer
